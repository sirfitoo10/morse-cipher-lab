# morse-cipher-lab
# ⚡ Morse Cipher Lab

> Encode. Decode. Communicate like a cyber agent.

A futuristic Morse Code encoder & decoder built with React + TailwindCSS.  
Designed with a hacker/cyberpunk aesthetic for fun projects and GitHub showcases.

---

## 🚀 Features

- 🔐 Encode text into Morse code
- 📡 Decode Morse back into text
- 📋 One-click copy output
- 💻 Cyberpunk / hacker-inspired UI
- 📱 Responsive design
- ⚛️ Built using React

## 📦 Installation

Clone this repository:

```bash
git clone https://github.com/sirfitoo10/morse-cipher-lab.git
```

Open folder:

```bash
cd morse-cipher-lab
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```
